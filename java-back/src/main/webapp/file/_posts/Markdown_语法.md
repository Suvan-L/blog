1.Markdown常用语法
======================

汇总
```

【换行】<br>
【空格】&nbsp;
【半方大缩进:】&ensp;`
【全方大缩进(Tab键盘)】: &emsp;
【粗体】 **内容**
【斜体】 _内容_
【加粗斜体】***内容***
【删除线】 ~~内容~~


标题(默认带有分割线,不同等级分割线粗细不同)
        【一级标题】============
        【二级标题】---------------


        【6级标题,字体大小逐级递减】
            #           1
            ##          2
            ###         3
            ####        4
            #####       5
            ######      6


列表
        【有序】
                1. content
                2. content
                3. content

        【无序(可用*,+,-)】
                + content
                + content
                + content


链接
        【文字】  [显示内容](URL链接)
        【图片】 
                <声明>  ![图片Alt][标记]
                <引用>  [标记]:图片地址 "图片Title"


【引用(可嵌套)】
        >
        >
        >
        >>>
        >>

表格
        |学号|姓名|
        |-|-|
        |1|皮皮虾|


```
