二.错误

1.桌面无法删除文件,用bat脚本强行删除！
-----------------------

```
<1>新建一个文本文档，写入（直接复制粘贴）下列命令：
　　del /f /a /q \\?\%1
　　rd /s /q \\?\%1
<2>另存为：删除ok.bat 
<3>把要删除的文件或者目录直接拖放到这个bat文件图标上就可以删除了
```
